# 🚽 带薪拉屎搭子 (ToiletBuddy)

> **“让每一次下蹲，都变成给资本主义的无情清算。”** 

这是一个专门为打工人量身定制的、幽默且具象化的**移动端优先 Web APP**。它能将你在工作期间上厕所的时间，精确到毫秒级地折算为实际赚取的带薪收益，并伴随有疯狂飞舞的 3D 金币动效和贴心的摸鱼阅读内容。

📈 **在线体验地址**：[带薪拉屎搭子（点击即刻开捞！）](https://yakinikudon.github.io/ToiletBuddy/)

---

## ✨ 核心亮点

### 💰 毫秒级带薪收益飞涨
- 支持**月薪 / 年薪**双模式输入。
- 采用 `requestAnimationFrame` 驱动高频数字跳动，根据薪资大小动态调整小数位数（最高展示 4 位小数），让你肉眼可见地感受数字飞涨的视觉爽快感！
- **真·后台运行**：记录绝对时间戳。无论你切回微信聊天、刷抖音还是锁屏，再次打开时收益绝对精准无误，绝不漏掉一分摸鱼钱。

### 🪙 Matter.js 2D 物理金币雨
- 集成专业的 `Matter.js` 物理引擎，真实还原金币落地堆叠与碰撞。
- **3D 翻转错觉**：金币在下落时会根据下落速度呈现正弦波式的 3D 翻转动效；落地静止后自动转化为侧面堆叠，实现令人惊叹的伪 3D 物理效果！

### 💭 沉浸式摸鱼小剧场与彩蛋
- **思考气泡**：拉屎小人的头上每隔数秒会随机冒出各种充满灵性的小心思（想加薪、发呆、想吃什么等）。
- **“尤里卡”灵感灯泡 💡**：每赚够 10 元，小人头顶会瞬间弹出金黄色的灵感爆棚灯泡，带给您满满的里程碑成就感！

### 📚 免后端实时 RSS 科技/AI 阅读器
- 深度整合免费公网 RSS 转换 API，实时抓取最新的**科技与 AI 前沿要闻**。
- 提供内置经典哲学书目（《被讨厌的勇气》、《沉思录》等经典片段）与最新网讯的混合列表。
- 支持一键“换一批”，并配备独立滚动的全屏高品质毛玻璃（Backdrop-Filter）深度阅读视窗，让您在马桶上汲取智慧。

### 🧾 扎心结算报告与防痔警示
- 根据您的拉屎时长和薪水进行趣味百分比判定，用大号黄金字体告诉您打败了全国多少摸鱼人。
- **健康关怀**：拉屎时间一旦超过 15 分钟，会自动弹出醒目的防痔疮警告，保护您的肛肠健康！
- **萌物点赞**：全新加入日系萌物「にっこリーノ」风格的扁平粑粑表情包，为您高超的摸鱼姿势竖起大拇指！👍

## 🛠️ 技术栈

- **构建工具**：Vite
- **前端框架**：React 18
- **物理引擎**：Matter.js
- **样式方案**：纯 Vanilla CSS（纯暗黑科技风设计，移动端完全自适应，无多余 UI 库负担）
- **数据源**：IT之家 RSS & 免费公网 RSS-to-JSON API

## 🚀 本地开发与部署

### 1. 克隆项目
```bash
git clone https://github.com/YakinikuDon/ToiletBuddy.git
cd ToiletBuddy
```

### 2. 安装依赖
```bash
npm install
```

### 3. 运行开发服务
```bash
npm run dev
```
打开浏览器访问控制台输出的地址（默认为 `http://localhost:5173/ToiletBuddy/`）即可。

### 4. 构建与部署到 GitHub Pages
项目已集成了 `gh-pages` 部署工具，只需一行命令即可自动完成打包并发布到您的 GitHub Pages 分支：
```bash
npm run build && npx gh-pages -d dist
```
